Temperature Monitor with Arduino

Beginner 45 minutes Arduino Uno + DHT22

Build a temperature and humidity monitor using an Arduino Uno and a DHT22 sensor. Readings are displayed in the Arduino Serial Monitor every 2 seconds.

📦 What You Will Need

  • Arduino Uno R3 or R4
    • USB cable (Type A to Type B)
    • Arduino IDE installed on your computer
  • DHT22 Temperature and Humidity Sensor
  • Breadboard and jumper wires
  • 10kΩ resistor (pull-up)

  1. Wire the DHT22 Sensor

    Connect DHT22 VCC to Arduino 5V, GND to GND and DATA to digital pin 2. Place a 10kΩ pull-up resistor between VCC and DATA.

  2. Install the DHT Library

    In Arduino IDE go to Sketch → Include Library → Manage Libraries. Search for "DHT sensor library" by Adafruit and install it along with the Adafruit Unified Sensor dependency.

  3. Upload the Code

    /* IoTopia – Temperature Monitor Tutorial */ #include <DHT.h> #define DHTPIN 2 // Data connected to pin 2 #define DHTTYPE DHT22 // Sensor model DHT dht(DHTPIN, DHTTYPE); void setup() { Serial.begin(9600); dht.begin(); Serial.println("IoTopia Temp Monitor Ready!"); } void loop() { float humidity = dht.readHumidity(); float temperature = dht.readTemperature(); if (isnan(humidity) || isnan(temperature)) { Serial.println("Error: sensor read failed!"); return; } Serial.print("Temp: "); Serial.print(temperature); Serial.print(" C | Humidity: "); Serial.print(humidity); Serial.println(" %"); delay(2000); // Read every 2 seconds }

  4. Open Serial Monitor

    Press Ctrl+Shift+M in Arduino IDE. Set baud rate to 9600. You should see temperature and humidity readings every 2 seconds.

💡 Pro Tip: The DHT22 is more accurate than the cheaper DHT11. Always wait at least 2 seconds between readings — the sensor needs time to stabilise.

Blink an LED – Hello World of Electronics

Beginner 15 minutes Arduino + LED

Blinking an LED is the "Hello World" of electronics. This is the simplest possible Arduino project and teaches you the basics of digital output control.

  1. Connect the LED

    Place an LED on the breadboard. Connect the longer leg (anode) through a 220Ω resistor to Arduino pin 13. Connect the shorter leg (cathode) to GND.

  2. Upload the Sketch

    /* IoTopia – Blink LED Tutorial Pin 13 has a built-in LED on most Arduino boards */ int ledPin = 13; // LED connected to digital pin 13 void setup() { pinMode(ledPin, OUTPUT); // Set pin 13 as output } void loop() { digitalWrite(ledPin, HIGH); // Turn LED ON delay(1000); // Wait 1 second digitalWrite(ledPin, LOW); // Turn LED OFF delay(1000); // Wait 1 second }

  3. Experiment with the Delay

    Change the delay values to make the LED blink faster (200ms) or slower (2000ms). This shows how software directly controls physical hardware.

💡 Next Step: Once mastered, try fading the LED with analogWrite() — this introduces Pulse Width Modulation (PWM), a key IoT technique.

ESP32 Wi-Fi Sensor Data Logger

Intermediate 90 minutes ESP32 + DHT22

Use the ESP32's built-in Wi-Fi to connect your sensor project to the internet. This is the foundation of a real IoT system — a device that sends data over a network.

  1. Set Up ESP32 in Arduino IDE

    Go to File → Preferences and add the ESP32 board manager URL. Then install the "esp32 by Espressif Systems" package via Tools → Boards Manager.

  2. Connect the Hardware

    Wire the DHT22 as before: VCC to 3.3V, GND to GND and DATA to GPIO pin 4 on the ESP32.

  3. Upload the Wi-Fi Sketch

    /* IoTopia – ESP32 Wi-Fi Data Logger */ #include <WiFi.h> #include <DHT.h> const char* ssid = "YOUR_WIFI_NAME"; const char* password = "YOUR_WIFI_PASSWORD"; #define DHTPIN 4 #define DHTTYPE DHT22 DHT dht(DHTPIN, DHTTYPE); void setup() { Serial.begin(115200); dht.begin(); WiFi.begin(ssid, password); Serial.print("Connecting to Wi-Fi"); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("\nConnected! IP: " + WiFi.localIP().toString()); } void loop() { float temp = dht.readTemperature(); float hum = dht.readHumidity(); Serial.printf("Temp: %.1f C | Humidity: %.1f %%\n", temp, hum); delay(5000); // Send every 5 seconds }

  4. View Live Data

    Open Serial Monitor at 115200 baud. Once connected you will see your device's IP address and live sensor readings every 5 seconds.

Smart Light with Raspberry Pi

Intermediate 60 minutes Raspberry Pi + PIR Sensor

Build a motion-activated smart light using a Raspberry Pi and a PIR sensor. The LED turns on automatically when someone enters the room — a real smart home feature.

  1. Connect the PIR Sensor

    Connect PIR VCC to Pi pin 2 (5V), GND to pin 6 (GND) and the OUT signal to GPIO 17 (pin 11). Connect an LED (with 330Ω resistor) to GPIO 27.

  2. Install the GPIO Library

    pip install RPi.GPIO

  3. Write and Run the Script

    # IoTopia – Smart Light with PIR Sensor import RPi.GPIO as GPIO import time PIR_PIN = 17 # PIR sensor signal pin LED_PIN = 27 # LED output pin GPIO.setmode(GPIO.BCM) GPIO.setup(PIR_PIN, GPIO.IN) GPIO.setup(LED_PIN, GPIO.OUT) print("IoTopia Smart Light – Waiting for motion...") try: while True: if GPIO.input(PIR_PIN): # Motion detected GPIO.output(LED_PIN, GPIO.HIGH) print("Motion detected! Light ON") time.sleep(10) # Stay on 10 seconds else: GPIO.output(LED_PIN, GPIO.LOW) time.sleep(0.1) except KeyboardInterrupt: GPIO.cleanup() print("Program stopped cleanly.")

  4. Test the System

    Run python3 smart_light.py from your terminal. Wave your hand in front of the PIR sensor — the LED will turn on for 10 seconds, then turn off automatically.

💡 Going Further: You can extend this project to send a notification to your phone using the Pushover API whenever motion is detected at home.
